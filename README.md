**AbilityWar** 기반으로 제작된 커스텀 능력 애드온입니다.

다양한 신규 능력과 유틸리티를 추가하여 기존 AbilityWar와는 또 다른 플레이 경험을 제공합니다.

---

# 📥 설치 방법

1. AbilityWar 플러그인을 설치합니다.
2. 서버를 한 번 실행한 뒤 종료합니다.
3. `plugins/AbilityWar/Addon` 폴더에 `BkueAddon.jar`를 넣습니다.
4. 서버를 다시 실행합니다.
