# # 블루애드온 (BkueAddon)
Daybreak님의 AbilityWar 커스텀 애드온 프로젝트입니다.

---

### ⚠️ 주의점 (Disclaimer)

* **비공식 2차 창작:** 본 플러그인은 Daybreak 님의 비공식 2차 창작 애드온 프로젝트입니다. 기반 프레임워크인 `AbilityWar` 및 관련 API, 모든 시스템 소스 코드의 저작권과 원본 소유권은 [Daybreak](https://github.com/Daybreak365) 님에게 있습니다.
* **환경 및 오류 책임:** 본 코드는 최적화 및 에러 방지를 최우선으로 고려하여 설계되었으나, 마인크래프트 서버 구동기(Paper, Purpur 등)의 특정 하위 빌드 버전, 자바 실행 환경, 혹은 타 플러그인과의 상호작용으로 인해 예기치 못한 에러가 발생할 수 있습니다. 개발자는 이로 인해 발생하는 게임 내 데이터 손실이나 서버 불안정에 대해 책임을 지지 않습니다.
* **이용 범위:** 본 애드온 코드는 개인적인 학습, private multiplayer 서버 플레이, 오픈소스 Fork 목적으로 자유롭게 수정 및 복사할 수 있으나, 이를 무단으로 재판매하거나 상업적인 이득을 취하는 행위는 엄격히 금지됩니다.

### ⚙️ 적용 방법 (How to Apply)
* AbilityWar 플러그인을 다운로드하여 서버의 `plugins` 폴더에 넣습니다
* 서버를 실행한 후 종료하거나 새로고침하여 필요한 폴더들을 생성합니다.
* `plugins/AbilityWar/Addon` 경로를 찾아서 이동합니다.
* 해당 폴더 안에 `BkueAddon.jar` 파일을 넣고 서버를 다시 실행합니다.
